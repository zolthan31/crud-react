Readme

Para Acesar a App,

abrir a pasta backend usando prompt de comando passando o comando npm start
apos sua inicialização pode visualizar o servidor de backend digitando na barra de endereço
o link http://localhost:3001/users

Abrir a pasta frontend usando prompt de comando passando o comando npm start
apos este passo abrira automaticamente o browser com a aplicação caso não digitar na barrade endereço o link http://localhost:3000

O App foi criado usando a biblioteca do Reacte e bootstrap mais media query para responsividade

Foi utilizado o Visual Studio como editor de texto
